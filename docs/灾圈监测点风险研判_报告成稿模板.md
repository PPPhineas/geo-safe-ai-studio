**占位符生成说明**（本节及以上「渲染占位符索引」标题起的内容在渲染时裁除，不入成稿；正文自 `# 灾圈地质灾害风险分析报告` 起进入成稿）：

- `point_list_rows`：代码按圈内点集生成表格行，重点点位「重点」列标注「★」；点位数超过阈值时清单转 CSV 附件，正文只内联重点点位。
- `chart_type_distribution` / `chart_warning_distribution` / `map_figure`：代码生成图件后以 Markdown 图片嵌入（`![key](url)`）；前端 `ChartInline.tsx` 拦截 `<img>` 标签，匹配组件 id 后替换为交互式 A2UI 图表（ECharts），无 A2UI 时降级显示静态 PNG。
- `a2ui_risk_card` / `a2ui_key_points` / `a2ui_recommendations` / `a2ui_text_limitations`：代码替换为 `![component_id](a2ui:component_id)` 引用，前端按 `a2ui:` 协议精确匹配 A2UI 组件，渲染交互式卡片/列表/文字。
- `研判.common_induce_factors`、`研判.recommendations.*`、`研判.data_limitations`：JSON 数组逐项渲染为 `- ` 列表行。
- `missing_value_summary`：代码侧缺失计数（如威胁人数缺失 N 条、坡度异常 M 条），渲染失败提示也会追加于此。
- `appendix_full_table`：代码按圈内点集输出全字段表；过长时转 CSV 附件。
- 正文不得再出现裸的占位符说明文字——面向开发者的注释一律写在本节，避免随成稿输出。
